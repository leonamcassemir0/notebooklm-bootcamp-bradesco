# 📘 Caderno de Estudos em Machine Learning com NotebookLM

## 📌 Contexto e Objetivos

Este projeto tem como objetivo a criação de um **caderno temático utilizando o NotebookLM**, focado no aprendizado de **Machine Learning**.

A proposta é utilizar a Inteligência Artificial como ferramenta de apoio para:
- Organizar conteúdos de forma estruturada
- Desenvolver pensamento crítico sobre o tema
- Criar materiais reutilizáveis para estudo contínuo
- Aprender na prática técnicas de **engenharia de prompts**

### 🎯 Objetivos de Estudo

- Entender os conceitos fundamentais de Machine Learning  
- Conhecer os principais tipos de algoritmos  
- Identificar aplicações práticas no mercado  
- Aprender a utilizar IA como ferramenta de aprendizado ativo  
- Criar um material consolidado para revisões futuras

## 🧠 Engenharia de Prompts e Aprendizados ("Cicatrizes")

Durante o uso do NotebookLM, foram realizados diversos testes de prompts para extrair melhores respostas da IA.

### 🔍 Exemplos de Prompts Utilizados

- "Explique Machine Learning como se eu fosse iniciante"  
- "Quais são os principais tipos de Machine Learning e exemplos práticos?"  
- "Resuma os principais pontos deste conteúdo em tópicos"  
- "Crie um glossário com os termos mais importantes deste material"  
- "Quais são aplicações reais de Machine Learning no mercado?"  

### 🔄 Variações Testadas

- Simplificação de linguagem  
- Respostas em formato de lista  
- Explicações com analogias  
- Pedidos de exemplos práticos  

### ⚠️ Dificuldades Encontradas

- Respostas muito genéricas quando o prompt não era específico  
- Falta de profundidade em perguntas amplas  
- Necessidade de refinar prompts para obter respostas mais técnicas  

### 🛠️ Soluções Aplicadas

- Tornar os prompts mais específicos  
- Pedir exemplos reais e aplicações práticas  
- Solicitar respostas estruturadas (listas, tópicos, resumos)

## 📖 Miniguia de Estudo (Entrega Final)

### 📌 Resumo Estruturado

**Machine Learning** é uma área da Inteligência Artificial que permite que sistemas aprendam a partir de dados, sem serem explicitamente programados.

#### 🔹 Tipos de Machine Learning

- **Supervisionado**  
  - Usa dados rotulados  
  - Ex: classificação e regressão  

- **Não Supervisionado**  
  - Trabalha com dados sem rótulos  
  - Ex: agrupamento (clustering)  

- **Aprendizado por Reforço**  
  - Baseado em tentativa e erro  
  - Ex: jogos e robótica  

---

### 📚 Glossário

- **Algoritmo**: conjunto de regras para resolver um problema  
- **Modelo**: representação treinada a partir de dados  
- **Dataset**: conjunto de dados utilizado no treinamento  
- **Treinamento**: processo de aprendizado do modelo  
- **Overfitting**: quando o modelo aprende demais os dados de treino  
- **Underfitting**: quando o modelo não aprende o suficiente  
- **Features**: variáveis utilizadas no modelo
